내 Vue.js + Node.js(Express) 기반 채팅 앱에서 사용자 검색 기능이 동작하지 않아.

현상은 다음과 같아:
- ChatView.vue에서 "username 또는 email로 검색" 입력창에 값을 넣어도 검색 결과가 안 나와.
- 브라우저 네트워크 탭에서는 `/api/users` 호출이 안 뜨거나, 검색어를 query 파라미터로 보내는 것 같아.
- 백엔드 Express에서는 `GET /api/users?search=xxx` 요청에 대응해 username 또는 email LIKE 검색을 해주는 코드가 있다.
- 검색이 제대로 작동하게 하려면 ChatView.vue의 Axios 요청 부분을 어떻게 고쳐야 할까?

아래 조건을 만족하는 코드로 수정해줘:
1. 사용자가 입력창에 입력하면 debounce되어 서버에 검색 요청을 보냄
2. 요청 시 query가 아니라 search 키로 파라미터를 전달해야 함
3. 결과는 users 배열에 저장되고 리스트로 렌더링됨
